# AWS S3 Event Notifications with SNS

This project demonstrates an event-driven cloud workflow using
AWS S3 and SNS.

## Objective
Trigger notifications automatically when objects are uploaded to an S3 bucket.

## Services Used
- AWS S3
- AWS SNS
- AWS IAM

## Workflow
1. Create S3 bucket
2. Configure event notifications on object upload
3. Create SNS topic and subscription
4. Assign IAM permissions
5. Verify notifications through test uploads

## Output
Successful SNS notifications received on file upload.

## Notes
This is a hands-on cloud architecture project.
